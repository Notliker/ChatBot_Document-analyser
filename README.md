# AI-ассистент для анализа и обработки документов на базе LLM и RAG с использованием Streamlit.

Этот проект разработан для загрузки, обработки и анализа PDF-документов (и других форматов, таких как DOC/DOCX). Система использует Retrieval-Augmented Generation (RAG) с Mistral AI для генерации точных ответов на основе содержимого документов. Проект поддерживает OCR для сканированных документов, векторизацию текста.

# Основные возможности (Архитектура и технологии)

1. Загрузка и обработка документов: Поддержка PDF, DOC, DOCX.
2. Преобразование в текст с помощью PyPDF и OCR (CLIP для изображений).
3. Векторизация и индексация: Использование HuggingFace Embeddings и Weaviate для хранения семантических векторов чанков.
4. RAG-система: Поиск релевантных фрагментов и генерация ответов с Mistral.
5. Интерактивный интерфейс: Веб-приложение на Streamlit для удобной работы с документами.
6. Метрики качества.

# Установка и запуск
1. Склонируйте репозиторий:
   
```
https://github.com/Badi2310/DocumentAnalyserAI.git
```

2. Перейдите в папку проекта:

```
cd DocumentAnalyserAI
```

3. Создайте файл `.env` в корне проекта и добавьте в него следующие переменные:
```
MISTRAL_API_KEY="ваш_ключ_от_Mistral"
WEAVIATE_API_KEY="ваш_ключ_от_Weaviate"
WEAVIATE_CLUSTER="ваш_кластер_Weaviate"
```

4. Соберите Docker-образ (Если Вы работаете на macOS, перед сборкой установите Docker Desktop, а также закомментируйте строки #6 и #7, т.к. не поддерживается GPU):
   
```
docker build -t my-streamlit-app .
```

5. Запустите контейнер:
   
```
docker run -p 8501:8501 my-streamlit-app
```

6. Откройте браузер и перейдите по полученному адресу (URL).

Теперь вы можете пользоваться ассистентом DocumentAnalyserAI.



# Использование
1. Загрузите документ: Выберите PDF/DOC/DOCX файл через интерфейс Streamlit.
2. Дождитесь загрузки документа (зеленого цвета).
3. Задайте вопрос по документу ассистенту. (Например: "Что говорится в этом документе?")
4. Получите ответ.
